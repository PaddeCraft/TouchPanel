OBS controller library WSv4

About

With this library you can automate Discord, e.g. mute your microphone or join a voice channel.

Setup

  1. Go to the Discord developer portal
  2. Create a new application
  3. In the sidebar, navigate to OAuth2
  4. Add a redirect to http://localhost
  5. Save Client ID and Client Secret, we'll need these later
  6. Create an action using one of the features of this library
  7. Run the previously created action
  8. Navigate to ~/.PaddeCraftSoftware/TouchPanel/
  9. Fill in the Client ID and Client Secret under the section "builtin.discord" in the file plugincfg.toml
  10. Open the Discord app on your computer, the website won't work
  11. Run the previously created action again
  12. Click 'Authorize' in the discord app
  13. You're ready to go!
